Scripting Documentation

BotBuilder uses a simple scripting language to automate actions. Each line must end with a | and no indentation is used.

Allowed Triggers

  • OnRun
  • OnMessage
  • UserJoin
  • OnBotMention

Allowed Functions

  • on("trigger", Variable) do
  • if <condition> then
  • elseif <condition> then
  • else
  • end
  • return <value>
  • function FuncName(value)

Allowed Operations

  • =
  • ==
  • +
  • -
  • *
  • /
  • and
  • or

Allowed Actions

  • SetRole(User, RoleName)
  • RemoveRole(User, RoleName)
  • Send(Channel, Message)
  • DM(User, Message)
  • Kick(User)
  • Ban(User)
  • Reply(Message, Text)
  • GetChannel(ChannelName)
  • HasRole(User, RoleName)
  • GetUserByName(Name)
  • Delay(seconds)
  • CreateChannel(Name, Category=None)
  • DeleteChannel(Name)
  • CreateCategory(Name)
  • DeleteCategory(Name)
  • Log(Message)
  • Random(min, max)
  • GetGuildName()
  • SetGuildName(Name)
  • React(Message, Emoji)
  • DeleteMessage(Message)
  • EditMessage(Message, NewText)
  • LockChannel(Channel)
  • UnlockChannel(Channel)
  • WaitForMessage(User, Timeout)
  • Mention(User)
  • PinMessage(Message)
  • UnpinMessage(Message)
